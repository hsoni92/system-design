# Google Project Management – Interview Prep

Interview-focused notes and examples from the **Google Project Management Professional Certificate** (Coursera). Use this folder to prep for PM, technical PM, or delivery/consulting interviews.

## How to use

1. **Syllabus** – [CURRICULUM.md](CURRICULUM.md) is the full course breakdown (7 courses, modules, skills, tools). Use it to see what’s covered and where each topic lives.
2. **By course** – Each numbered file below is one course. Open the course that matches the topic you’re prepping (e.g. initiation → 02, Agile/Scrum → 05).
3. **Per file** – You get: **key terms** (definitions), **worked examples** (SMART, RACI, WBS, risk, user stories, etc.), **interview Q&A** (short answers), and **“How would you…”** prompts for behavioral/STAR.
4. **Before the interview** – Skim key terms and one example per artifact; practice 2–3 Q&A and one “walk me through a project” (see [06-Capstone-Real-World.md](06-Capstone-Real-World.md)).

## Quick reference

- **Frameworks:** SMART, OKRs, RACI, WBS, Critical Path, Triple Constraint, ROAM, PDCA, Tuckman, STAR  
- **Artifacts:** Charter, risk register, communication plan, SoW, backlog, status report, closeout  
- **Concepts:** Waterfall vs Agile, Scrum roles/events/artifacts, three-point estimating, story points
